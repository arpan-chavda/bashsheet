Search the content of all the files(Slow method)
locate [text]
Search for file(Slow method)
locate [file_name]
Update database of files
sudo updatedb
Find file(Fast method)
find -name "[file_name]"
Find files who start with word text
find -name "[text]"
Find files who end with word text
find -name "*[text]"
Search from file Size (in ~)
find ~ -size +10M
{search files bigger than.. (M,K,G)}
Search from last access
find -name "[file_type]" -atime -5
{'-' = less than, '+' = more than and nothing = exactly}
Search only files or directory
find -type f
find -type d
