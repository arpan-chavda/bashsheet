Info
Search for text in file
grep [text] [file_name]
{-i = Doesn't consider uppercase words
-I = exclude binary files}
Search for file names with occurence of the text
grep -r [text] [dir_name]
Search start of lines with the word text
grep -E [text] [file_name]
Show lines containing numbers 0-4
grep -E [0-4] [file_name]
Retrieve all lines with alphabetical letters
grep -E [a-zA-Z] [file_name]
Sort the content of files
sort [file_name]
Write result to a file
sort -o [source_file] [output_file]
Sort in reverse,randomly and numbers
sort -r [file_name]
sort -R [file_name]
sort -n [file_name]
Count words in file
wc [file_name]
{Other Options: -l (lines), -w (words), -c (byte size), -m (number of characters)}
Cut a part of file
cut -c 2-5 names.txt
{It cut the characters 2 to 5 of each line}
Other options: -d (delimiter), -f (# of field to cut)
