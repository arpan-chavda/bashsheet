Info
View current date and time of system
date
Options with date command
date +%H   {shows hour}
date +%M   {shows minutes}
date +%S   {shows seconds}
For more info: man date
Execute program at another time
at [time_of_execution] [ENTER]
[ENTER_COMMAND] [ENTER]    {Repeat this step if you want more}

CTRL+D to close input
Show a list of jobs waiting to be executed
atq
To remove a job
atrm [no_of_job]
Pause between commands
sleep
touch file; sleep 10; rm file  {Example}
Execute a command regularly
crontab
Options: -e = modify the crontab
-l = view current crontab
-r = delete you crontab
Syntax of crontab:
[Minutes] [Hours] [day_of_month] [day_of_week(0-6, Modify: 0 = Sunday)] [COMMAND]
Execute command in background
Put ;& at the end of command like cp *.txt &
{'-' = less than, '+' = more than and nothing = exactly}
